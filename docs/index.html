Features

Everything you need to manage Claude Code

Dashboard

Overview stats, storage breakdown, recent activity timeline, and config summary at a glance.

Plugin Manager

Enable, disable, view docs, or uninstall plugins. Grouped by source with search filtering.

Session Manager

Browse conversations, rename sessions, read full content with message bubbles, and copy resume commands.

Config Editor

Edit settings.json, CLAUDE.md, and permissions directly in the browser with structured forms.

MCP Servers

View, enable/disable, and delete MCP servers across all plugins. See type, URL, and auth status.

Skills Browser

Manage installed skills with enable/disable/delete. Browse the marketplace with install status indicators.

Cleanup Tool

Free disk space with selective cleanup. Open folders in file explorer for manual management.

System Info

Token usage by model, 24-hour activity heatmap, daily statistics, and full usage history.

i18n & Themes

Full English and Chinese support with one-click switch. Dark mode and light mode toggle.

Quick Start

Up and running in 30 seconds

Terminal
$ git clone https://github.com/ryan11njr/cc-config-ui.git
$ cd cc-config-ui
$ python server.py
# Open http://127.0.0.1:8787 in your browser
